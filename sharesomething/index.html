记录一些人和事

2023/5/2

  • 一天半把辉夜三集+剧场版重温了一遍，起初剧场版不理解会长，后来虽然还是做不到完全理解（毕竟前三季的会长太帅了）但也算是稍有共情之处。

  • 我打算一阵子不看动漫，看的时候虽然很享受，可那之后就是无尽的空虚感袭来，那感觉更甚于游戏，（游戏戒掉一个月）。

2023/5/3

  • 还是不在状态，怎么办怎么办，想要摆烂，但是不知道怎么摆，除了学习不知道干什么，又学不进去！！我喜欢他看起来很凶的眼神，疲惫时努力的象征！！——辉夜
  • 总结一下，自4/15日开始瑞吉外卖的多表操作之后，所展示的低迷和不在状态，导致了长达半个多月的复杂情绪阶段，直至今日不仅没有丝毫消退的痕迹，反而愈演愈烈，在辉夜的完结中到达了高潮。
  • 原因就是，学习不再是我心中所思考的第一要务，相反，在学习的时候应对懈怠产生的情绪才是这半个多月来，一直隐藏的魔鬼，这也是我越来越起不来，越来越没有拼劲的原因
  • 重新找到学习的乐趣是当务之急！！五月份一个月不碰游戏+认真学习，六月初，奖励自己 做一次五百元以内的按摩，（一个月的拼搏换来一天的享受我认为是金钱难以衡量的）

2023/5/7

  • 4号青年节，错过了一包牛奶。血亏，我都已经快忘记自己是个青壮年了…..
  • 周五周六开摆！结果周六本来被拳头骗已经够了，还被换肤制裁，号没了，体验为 非常差！
  • 今天去看了牙齿，让我又找到了一个不得不奋斗赚钱的理由，忽然有了动力，之前的负面消极情绪又没了，我猜测有如下几条原因：
    1. 跟上次查出眼睛有病一样，有迫切感，想减轻家庭的负担，迫切的想要挣钱上班，自己给自己负担医药费
    2. 最近reggie项目结束，找到了新的学习目标并且已经开始行动，且一开始内容都是学过的比较简单学起来没啥压力
    3. 对人生的理解又更进一步了吧，越来越注重自己，越来越不在意外界对自己的干扰。（可能有无意间见到了心心念念的前女友一面的原因，她已经前进了。像一朵绽放的花朵，她散发着自信和坚定，在旧情的回忆中寻找新的方向。祝福她在前行的路上，每一步都充满欢笑和收获。）
    4. 天气..变好了开始变得乐观积极向上

2023/5/9

情绪波动严重，夏天到了……….问了gpt一个问题，我觉得与我所期待的答案类似，值得记录并行动！

要专注于提升自我，需要掌握以下几个方面：

  1. 制定目标：在生活中多思考自己想要实现的目标和愿望。制定明确的目标能帮助你更加专注于自我提升，从而避免被身边事物的干扰。
  2. 学习新技能：通过学习新技能来增强自己的能力和知识储备，这样可以让你更有自信心，更具有抵御外部干扰的能力。
  3. 独立思考：避免盲目跟从他人的看法和思想，要有自己独立思考的能力。可以通过阅读、思考等方式，培养独立思考的习惯，这样才能更好地为自己制定出最适合自己的发展计划。
  4. 定期反思：对自己的成长进行反思，了解自己的成长和短板所在。通过这种方式，可以不断改进自己的行为，并找到自己的优缺点以及应该改变的方面。
  5. 保持积极心态：随时为自己打气，阳光和积极的思维方式会激励自己积极面对生活中的困难和挑战。这样，你就不容易被身边的事物所干扰。

总之，对于提升自我而言，关键在于自我认知和坚定信念。当你知道自己想要什么，并注意自由、积极面况便的时候，你别无选择但去关注和追寻自己的目标

  • 阅读《Head First Servlets 和 JSP》这本书，并理解其中的 Web 编程原理；
  • 学习 HTTP 协议和服务器相关知识。

  • 先通过使用 Spring Initializr 创建一个简单的 Spring Boot 项目来入门；
  • 学习 Spring Boot 相关注解和配置，例如 @SpringBootApplication、@RestController 等；
  • 深入了解 Spring Boot 自动装配、切面编程、Web 开发等；
