6.平方根问题

这个问题主要是在思路：

分析：录入一个值，我们定义为num，下一步是计算出num的平方根，结果只保留整数部分怎么实现，再定义一个变量i，i的平方跟num作比较，大于num，则i-1就是整数部分，等于num则i就是num的平方根。

7.质数问题

思路：判断一个数num是否为质数，定义一个从2到num范围内的循环，条件是，如果该范围内的每一个数字都不能被num整除则num是质数。

flag思想

定义一个flag作为是否为质数的一个标记，遍历所有的数之后，有能被i除进的数则flag变为false不是质数，如果没有则是，这也决定了最后面判断是否为质数的if语句要等循环结束，所有的数字都找完之后，再做判断。

优化：二分法，

原理：如81的平方根 9，则a* b=81 那么a和b中必有一个是小于等于9，另一个是大于等于9

9.数组的两种初始化方式

静态初始化：int arr []={1,2,3,4,5}

动态初始化：int arr [] =new int[5]

静态初始化：手动指定数组的元素，系统会根据元素的个数，计算出数组的长度。

动态初始化：手动指定数组长度，由系统给出默认初始化值。

4.成员变量和局部变量

  1. 类中位置不同，前者类中方法外，后者方法内或者方法声明上
  2. 内存中位置不同，成员变量（堆内存），局部变量（栈内存）这样理解：栈内存小，方法中的局部变量，随着方法的使用而存在，方法使用完之后出栈减少空间消耗。
  3. 生命周期：成员变量随着对象的存在而存在，局部变量随着方法的调用而存在
  4. 初始值不同：成员变量有默认初始化值，局部变量没有，必须先定义，赋值才能使用
